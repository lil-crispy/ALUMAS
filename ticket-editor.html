Editor Termico ALUMAS

Editor libre tipo plantilla para crear tickets termicos con texto e imagenes y exportarlos para impresion o para integracion TSPL.

Configuracion

La plantilla se diseña sobre una sola etiqueta y luego se repite en 3 columnas por fila.

La cantidad siempre se ajusta automaticamente al multiplo de columnas para que cada fila salga completa.

Herramientas

Propiedades

Selecciona un bloque para editar su contenido, tamaño, alineacion o borde.

Exportacion

El archivo TSPL se genera en binario con comando BITMAP, listo para enviarlo por Python con win32print.

Uso rapido

  • Haz clic en un bloque para seleccionarlo.
  • Arrastra el bloque para moverlo dentro de la etiqueta.
  • Usa el punto naranja de la esquina para redimensionarlo.
  • Doble clic en un texto para editarlo directo en el lienzo.
  • La tecla Supr elimina el bloque seleccionado.

Plantilla libre para tickets termicos

La edicion funciona como una hoja visual: puedes escribir, borrar, subir logos o imagenes, reorganizar elementos y luego repetir el ticket automaticamente en la matriz de impresion.
